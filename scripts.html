Node Scripts

Notes

  • npm scripts are found in your project's package.json file
  • npm custom scripts must be preceded by either run-script or run
"scripts": {
	"test": "echo \"Error: no test specified\" && exit 1",
	"test-msg": "echo \"Exit Test\" && exit 0",
	"start": "node getDrugList"
}

## npm start ##

When issuing the 'npm start' command node will look for the key 'start' in the scripts object of package.json.

	If found it will run the command value specified.
	If not found node will run 'node server.js'
		If server.js does not exists node will write an error to the console:
			npm ERR! missing script: start

## --silent ##

Adding this flag sets the log level to silent. It will suppress log output to the console.

Or globally by creating a .npmrc file(this file can be either in your project directory or your home folder) with the following:

loglevel=silent
